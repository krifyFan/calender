## 自由选择模式
![images](https://github.com/krifyFan/image-folder/blob/master/free.png)<br>
1、使用原生js写的日历插件<br>
2、支持自定义选中日期<br>
3、支持自定义周期选中日志，比如从1号到20号，每隔3天选中日期<br>
4、支持星期的选择，比如勾选星期一，自动选中当前月份的所有星期一<br>
